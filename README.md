# 🍺 Ranking Piw

Prosta apka-ranking piw. **Bazą danych jest plik `Piwa.xlsx`** — żeby dodać piwo,
ocenę albo komentarz, wystarczy edytować Excela i wgrać go do tego repozytorium
(np. przeciągnij plik na stronie repo → "Add file" → "Upload files"). Strona
zaktualizuje się sama po ok. 1 minucie.

Kolumny w Excelu (arkusz `Ocenka`): `lp`, `nazwa`, `%`, `rodzaj`, `OCENA` (0,25–5),
`uwagi/komentarze`, `link`.

Strona działa na GitHub Pages — dostęp ma każdy, kto ma link.
